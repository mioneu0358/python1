주요기능: 매 시험 점수를 입력해서 자신이 원하는 대학교의 커트라인과 비교 후, 
어떤 과목에 더 집중을 해야 할지 피드백을 주는 형태

유저: 매 기간 시험점수 => 개인 디비형태로 구현
서버: DB는 학교별 커트라인 및 입시 정보,

과목별 합격 %가 가장 떨어지는 순으로 출력해준다.
=> 숨.고나 에듀윌/메가스터디/인터넷강의/ 
해당 학교 재학생들 과외연결(1~3학년) or 상담 

DataScience
pandas - DB조작, 검색	read_
numpy -  수학관련 함수식
matplotlib - 그래프,도표 그리기

나의 성적 입력하기 -> 대학정보 엑셀파일 불러오기 -> 내 성적과 대조하기 -> 차이값 출력하기
